# text-to-speech.html

Browser-based, free text-to-speech tool. No character limit, no setup, no internet connection required.

## Usage

1. Download `text-to-speech.html`
2. Open it in your browser
3. Paste your text and hit play

CSS and JavaScript are embedded in the file, no external dependencies.

## Features

- Web Speech API based, no data is sent to any server
- Voice, speed, pitch and volume controls
- Automatic sentence chunking for long texts (workaround for Chrome's cutting bug)
- Theme preference and settings are saved to localStorage
